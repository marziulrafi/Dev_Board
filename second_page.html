What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML returns all text, including html tags, that is contained by an element.
2. innerText returns all text contained by an element and all its child elements.
3. textContents is all text contained by an element and all its children that are for formatting purposes only.

What is event delegation in the DOM?

Event delegation is a DOM technique where a single event listener is attached to a parent element to handle events from its child elements, instead of attaching multiple listeners to each child

What is event bubbling in the DOM?

Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM. It allows parent elements to respond to events triggered by their child elements.
